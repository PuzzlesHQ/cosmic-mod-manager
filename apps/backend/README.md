## API Endpoints

- **Auth**: OAuth providers, session management, personal access tokens
- **Users & Organizations**: profiles, teams, roles
- **Projects & Versions**: full CRUD for mod projects, versions, and version files
- **CDN**: file/asset serving
- **Search**: full-text search with filtering
- **Moderation**: content reports and mod approval queue
- **Collections**: user curated collections
- **Threads**: report and moderation chat threads
- **Notifications**: team invitation and project status change notifications
- **Sitemap**: auto-generated XML sitemaps (for the frontend)
